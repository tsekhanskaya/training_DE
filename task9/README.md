# Task 9. Alert project

## Preparing
1. Set up your email to be notified in the message view.
2. Add .env with next values:
```
SENDER_EMAIL=
SENDER_PASSWORD=
RECEIVER_EMAIL=
```

## Run

1. ```docker compose up --build```
2. Add log file to data folder.
